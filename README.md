# Apiiro Homebrew Tap

Homebrew tap for Apiiro CLI tools.

## Installation

```bash
# Tap and install
brew tap apiiro/tap
brew install apiiro
```

## Usage

After installation, run:

```bash
apiiro --help
apiiro --version
```

For more information, contact [Apiiro](https://apiiro.com/).
